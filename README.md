# ConectaTurbo 

Plataforma de alfabetización digital desarrollada con PHP, MySQL, HTML, CSS y Bootstrap.  
Proyecto enfocado en comunidades rurales de Turbo, Antioquia.

---

##  Descripción

ConectaTurbo es una plataforma web que permite:

- Registro de usuarios
- Inicio de sesión seguro
- Visualización de cursos
- Evaluación básica de conocimientos
- Gestión de datos mediante base de datos MySQL

---

## Tecnologías utilizadas

- PHP
- MySQL (MariaDB)
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- XAMPP (servidor local)
